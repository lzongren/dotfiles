# MarkEdit

Configuration and extensions for [MarkEdit](https://github.com/MarkEdit-app/MarkEdit).

## Extensions

### [MarkEdit-preview](https://github.com/MarkEdit-app/MarkEdit-preview)

Markdown preview with Mermaid, KaTeX, and syntax highlighting.

**Install / update:**

```bash
./markedit/install.sh
```

Then restart MarkEdit to apply the changes.

The script always fetches the latest release from GitHub, so running it again acts as an update.
